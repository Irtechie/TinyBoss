# TinyBoss Installer

Builds and packages the self-contained TinyBoss installer.

## Build

From the repo root:

```powershell
.\Installer\Build-Installer.ps1
```

Output:

```text
Installer\Output\TinyBoss-Installer
```

Run:

```text
Installer\Output\TinyBoss-Installer\TinyBoss.Installer.exe
```

## What The Installer Does

- Publishes TinyBoss as a self-contained Windows app.
- Copies app files to `%LOCALAPPDATA%\Programs\TinyBoss`.
- Installs shortcuts.
- Registers uninstall metadata.
- Creates the `TinyBoss Elevated Startup` scheduled task.

## Uninstall

The packaged app includes:

```text
uninstall-tinyboss.ps1
```

It stops TinyBoss, removes installed files, removes shortcuts, removes uninstall
metadata, and unregisters the elevated startup task.

## Development Notes

- Keep install steps idempotent.
- Do not recursively delete paths unless they resolve under the intended install
  directory.
- The app bundle under `TinyBoss.Installer\app` is generated by the build script.
